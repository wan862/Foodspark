About

This project was developed as a Online Food Ordering System by
Wan Runhao, Jason Zeng Xianyu, Wang Bingqi, Wai Jit

This website provides students sign up.
The student can see various menus provided by the restaurants and order anything.

Student orders shows up on the dashboard, and can be submitted or cleared.

This website is a stand-alone complete product, and can be used commercially.
